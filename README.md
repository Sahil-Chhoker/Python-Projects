## How to Get Started

### Clone the Repository

```bash
git clone https://github.com/sahil-chhoker/python-projects.git
cd python-projects
```

### Set Up Virtual Environment

```bash
python -m venv venv
source venv/bin/activate  # For Windows: .\venv\Scripts\activate
```
